## 7. Integration & Validation Skills

### Skill: End-to-End System Validation
**Description**
Ability to reason about and validate the system across all layers.

**Capabilities**
- Frontend ↔ Backend ↔ Database validation
- Auth flow verification
- Cross-user isolation testing

**Used By**
- Integration Tester Agent

---

### Skill: Spec-to-Implementation Auditing
**Description**
Ability to detect deviations between specs and implementation.

**Capabilities**
- Compare behavior vs acceptance criteria
- Identify missing features
- Recommend corrective actions

**Used By**
- Integration Tester Agent
