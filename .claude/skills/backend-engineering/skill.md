## 5. Backend Engineering Skills

### Skill: FastAPI Development
**Description**
Ability to build RESTful APIs using FastAPI following best practices.

**Capabilities**
- Route creation
- Dependency injection
- Request/response validation
- Error handling with HTTPException

**Used By**
- Backend Engineer Agent

---

### Skill: API Contract Enforcement
**Description**
Ability to strictly implement APIs as defined in specs.

**Capabilities**
- Follow endpoint definitions exactly
- Respect request/response schemas
- Prevent undocumented behavior

**Used By**
- Backend Engineer Agent
- Integration Tester Agent
