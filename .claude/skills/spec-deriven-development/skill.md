## 1. Spec-Driven Development Skills

### Skill: Spec-Kit Authoring
**Description**
Ability to write structured, implementation-ready specifications using Spec-Kit Plus conventions.

**Capabilities**
- Write feature specs with user stories & acceptance criteria
- Organize specs into features, api, database, ui
- Reference specs using `@specs/...`
- Maintain consistency across specs

**Used By**
- Spec Writer Agent
- Architecture Planner Agent
- Integration Tester Agent

---

### Skill: Requirements Decomposition
**Description**
Ability to break high-level requirements into atomic, testable specifications.

**Capabilities**
- Identify features, constraints, and edge cases
- Convert requirements into measurable acceptance criteria
- Avoid implementation assumptions

**Used By**
- Spec Writer Agent
