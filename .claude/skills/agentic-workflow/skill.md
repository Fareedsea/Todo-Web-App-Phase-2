## 8. Agentic Workflow Skills

### Skill: Agent Boundary Discipline
**Description**
Ability to operate strictly within agent responsibility boundaries.

**Capabilities**
- No premature coding
- No cross-agent role leakage
- Respect execution order

**Used By**
- All Agents

---

### Skill: Spec-First Enforcement
**Description**
Ability to prevent implementation before specs are approved.

**Capabilities**
- Reject incomplete specs
- Enforce workflow order
- Maintain project integrity

**Used By**
- Program Manager Agent
- Integration Tester Agent

---
